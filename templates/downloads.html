Bulk Downloads

{% rest %} Because it is often convenient to get all of a state's data for testing or offline use we provide bulk downloads of our data. We currently offer bulk data in two formats, JSON and CSV. JSON Archives ============= The data contained in these downloads is in the same format as that obtained via the `API `_, refer to those docs for specifics on file formats. These downloads are not intended as a replacement for using the API, the main deficit you'll notice is that they lag up to a month (but usually not more) behind the current data. This makes them useful for general research purposes or initializing a database that will then be kept up to date via regular calls. If you'd like more current data please use the `API `_. One last thing, **these files are large**. They are often a couple years worth of legislative data and generally fall in the 100MB range. CSV Archives ============ Because not everyone is comfortable working with JSON data we are now offering **experimental** CSV files. `Documentation on the CSV bulk files `_. Like the JSON archives, these aren't intended as a replacement for the API.
