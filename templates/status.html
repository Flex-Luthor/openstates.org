State Status

States are considered to be "ready" or "experimental" depending on how reliable the source data is and how thoroughly we've vetted and cleaned up the scraped data. We do our best to provide high quality data for all states, but you are more likely to find incomplete data where we haven't been able to identify a source or done manual reconciliation of ids when using experimental states.

If you are curious about the specific status of state data look at detailed status below or feel free to ask on the Open States google group.

If you encounter issues using data from any state please let us know via the Open State Project Issue Tracker.

Unsupported States

The remaining states that are grey are not yet in our API. If you are interested in contributing visit the Contributor's Guide that will guide you through obtaining the source and speaking with project developers that can help you determine what work needs to be done for the state of your choosing.
