Project Status

As of March 1st, 2011 we have data for 20 states and the District of Columbia.

States are denoted as either "ready" or "experimental" depending on how reliable the source data is and how thoroughly we've vetted and cleaned up the scraped data. We do our best to provide high quality data for all states, but you are more likely to find incomplete data where we haven't been able to identify a source or done manual reconciliation of ids when using experimental states.

If you are curious about the specific status of state data check below for any state specific issues or ask on the Open States google group.

If you encounter issues using data from any state please let us know via the Open State Project Issue Tracker.

State Status

Ready States

Ready states (green on the above map) are those that we've identified as having the highest quality data. Currently this includes California, Louisiana, Maryland, Minnesota, New Jersey, North Carolina, Texas, Utah, Vermont, and Wisconsin.

Experimental States

Experimental states (in blue) are those that we support but have not yet promoted to ready status. Alaska, Arizona, Florida, Mississippi, Nevada, Ohio, Pennsylvania, South Dakota, Virginia, Washington and Washington DC fall in this category.

Unsupported States

The remaining states (gray on the map) are not yet in our API. If you are interested in contributing visit the Contributor's Guide that will guide you through obtaining the source and speaking with project developers that can help you determine what work needs to be done for the state of your choosing.

State-specific Issues

California
  • At this time, we are collecting California's data from a provided SQL database and are unable to provide standard source URLs for legislators.

District of Columbia
  • DC does not have a uniform source for committee memberships and so we currently do not provide committee information.

Minnesota
  • Minnesota does not provide legislator roll calls for votes taken in the Senate so we are lacking detailed votes for the MN Senate.
